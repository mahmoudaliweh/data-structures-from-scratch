# Queue



Queue is a linear data structure that allows insertion at the back and deletion from the front, it enforces FIFO First-in First-out ordering.

Internally it could be implement as an array or a linked list.



# Why Queue?



It enforces FIFO ordering of the data.



# Applications



OS process scheduling

Web servers: request handling queues

Job Queues



# Operations Time Complexity



|Operation|Time Complexity|
|-|-|
|Deletion at front|O(1)|
|Insertion at back|O(1)|
